Cloud Portfolio Website

LIVE

Designed and deployed this portfolio website entirely on AWS. The site is stored in Amazon S3, served globally through a CloudFront CDN with edge caching, secured with a free TLS certificate via AWS Certificate Manager, and accessible via a custom domain managed in Route 53. IAM policies control access between services following the principle of least privilege.

Architecture

Route 53 → CloudFront (SSL/TLS) → S3 (Origin)
